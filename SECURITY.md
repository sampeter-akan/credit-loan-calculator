# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in this project, please report it privately to the project maintainer:
- Email: samuelakanimo@hotmail.com
- GitHub: https://github.com/sampeter-akan/credit-loan-calculator

Do not disclose security issues publicly until they have been reviewed and addressed.

## Security Best Practices
- No secrets, credentials, or sensitive data should be committed to the repository.
- Use environment variables or CI/CD secret stores for all credentials.
- Keep dependencies up to date and monitor for vulnerabilities (e.g., with Dependabot).
- Limit access to the repository and CI/CD systems to trusted users.
- Regularly review and clean up repository files to avoid exceeding storage limits and leaking sensitive data.

## Supported Versions
Only the latest version of the project is actively maintained and receives security updates.

## Further Guidance
For more information or to contribute securely, see the README.md or contact the maintainer.
